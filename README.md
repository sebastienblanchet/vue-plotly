# Vue-plotly

Forked `/example` from [this repo](https://github.com/David-Desmaisons/vue-plotly).

## Client

Start GraphQL server:
```bash
cd client
npm install && npm run apollo
```

Start Vue.js app:
```bash
cd client
npm install && npm run serve
```

## GraphQL Schema

[localhost:4040/graphql](http://localhost:4000/graphql)
